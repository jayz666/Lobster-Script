# Item Images

This folder contains placeholder images for the lobster fishing items. For a complete installation, replace these with actual 64x64 PNG images:

- `lobster.png` - Regular lobster item image
- `rare_lobster.png` - Rare lobster item image  
- `lobster_trap.png` - Lobster trap item image
- `bait.png` - Fish bait item image

The script will automatically create placeholder entries, but you should add real images for the best user experience.
